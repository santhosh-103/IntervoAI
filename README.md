# 🚀 IntervoAI – AI Powered Mock Interview Platform

IntervoAI is a full-stack AI-powered mock interview platform built using the MERN Stack.  
It helps students and job seekers practice technical, HR, voice, video, and coding interviews in a real interview environment.

# ⚙️ Installation

## 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/intervoai.git
```

---

## 2️⃣ Install Frontend Dependencies

```bash
cd client
npm install
```

---

## 3️⃣ Install Backend Dependencies

```bash
cd ../server
npm install
```

---

# 🔑 Environment Variables

Create `.env` inside `server/`

```env
PORT=5000

MONGO_URI=your_mongodb_connection

JWT_SECRET=your_secret_key

CLIENT_URL=http://localhost:5173
```

---

# 🚀 Run Project

## Backend

```bash
cd server
npm run dev
```

---

## Frontend

```bash
cd client
npm run dev
```
